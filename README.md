

# Pomodoro Timer

This android application for studying or working with pomodoro technique



## Authors

- [@blblblblblblblblbl](https://github.com/blblblblblblblblbl)


## Features

- timer
- timer configuration
- history
- day
- settings
- metrial theme for color picker



## tech stack
- Android studio
- Kotlin
- Clean Architecture
- multi-module app
- Coroutines 
- Flow
- OOP
- SOLID
- Jetpack Compose
- Jetpack Navigation
- Material Design 3
- Material You
- Worker
- Broadcast receiver
- Room
- Hilt
- Unit,UI testing
- Mockito
